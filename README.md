# Student Management System (Java + Swing + JDBC + MySQL)

A complete Student Record Management System built using:
- Java
- Java Swing (GUI)
- JDBC
- MySQL

This project supports:
- Admin Login System
- Multiple Admins
- Add / View / Update / Delete Students
- Grade & Pass/Fail Calculation
- Search Students by Name
- Filter Students by Year & Course
- Desktop GUI Interface using Swing

## Technologies Used
- Java (JDK 17+)
- Java Swing
- MySQL 8
- JDBC (MySQL Connector/J)

## Project Features
- Secure admin authentication
- Multi-admin support
- Real-time database operations
- Grade and status calculation based on marks
- Search and filter functionality
- GUI built with Swing

## How to Run the Project
1. Start MySQL server
2. Create database studentdb
3. Create tables students and admins
4. Insert at least one admin user
5. Add MySQL Connector JAR to lib
6. Run GUIApp.java
